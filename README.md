# 🚀 Diginatize | Premium Digital Marketplace

![Diginatize Header](https://raw.githubusercontent.com/lucide-react/lucide/main/icons/globe.svg) 

**Diginatize** is a high-end, immersive digital product marketplace designed with a "Future-First" aesthetic. It features interactive 3D elements, smooth scroll-driven animations, and a luxury glassmorphism UI.

## ✨ Key Features

- **🌐 Interactive 3D Globe**: A custom-built WebGL globe that responds to user scroll, providing an immersive navigation experience.
- **🤖 3D Robot Mascot**: Integrated Spline 3D scene featuring an interactive robot character that follows the user's cursor.
- **💎 Premium UI/UX**: Built with React and Tailwind CSS, featuring sleek gradients, micro-animations, and a modern glassmorphism design system.
- **📱 Fully Responsive**: Optimized for all devices, from high-end desktop monitors to mobile screens.
- **🚀 High Performance**: Built with Vite for lightning-fast load times and optimized asset delivery.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **3D Engine**: [Spline](https://spline.design/) & [Three.js](https://threejs.org/)
- **Animations**: [GSAP](https://greensock.com/gsap/) & Framer Motion
- **Icons**: [Lucide React](https://lucide.dev/)

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/diginatize.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🚀 Deployment

This project is configured for **GitHub Pages** via GitHub Actions.

1. Push your code to the `main` branch.
2. Go to your GitHub Repository **Settings** > **Pages**.
3. Under **Build and deployment** > **Source**, select **GitHub Actions**.
4. Your site will be automatically built and deployed!
